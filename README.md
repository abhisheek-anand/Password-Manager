# Password Manager

A Password Manager built using Python and Object-Oriented Programming (OOP).

## Features

* Add Password
* View Passwords
* Search Passwords
* Update Passwords
* Delete Passwords
* Password Strength Checker
* Statistics Dashboard
* Binary File Storage using Pickle
* Duplicate Website Detection

## Concepts Used

* Classes and Objects
* Constructors
* Methods
* Lists of Objects
* Binary File Handling
* Exception Handling
* Searching and Updating Records

## How to Run

1. Install Python 3
2. Download the project
3. Open terminal in project folder
4. Run:

```bash
python password_manager.py
```
